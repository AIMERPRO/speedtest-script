# speedtest-script

CLI-скрипт для замера скорости интернета. Выполняет N последовательных HTTP-запросов к указанному адресу и выводит среднюю скорость загрузки в МБ/с.

## Как это работает

1. Отправляет N последовательных GET-запросов к указанному URL.
2. Для каждого запроса замеряет время и объём полученных данных.
3. Выводит результаты каждого запроса и итоговую статистику:
   - среднее время запроса
   - суммарный объём скачанных данных
   - среднюю скорость в МБ/с

## Требования

- Python 3.9+

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python main.py <URL> [--count N]
```

### Примеры

```bash
# 10 запросов (по умолчанию)
python main.py http://speedtest.tele2.net/10MB.zip

# 5 запросов
python main.py http://speedtest.tele2.net/10MB.zip --count 5
```

## Аргументы

| Аргумент  | Описание                                        | По умолчанию |
|-----------|-------------------------------------------------|--------------|
| `url`     | URL для запросов (обязательный)                 | —            |
| `--count` | Количество последовательных запросов            | `10`         |

## Пример вывода

```

Запросы: 10
--------------------------------------------------
  [ 1/10]  0.312 с   5.84 МБ
  [ 2/10]  0.287 с   5.84 МБ
  ...
  [10/10]  0.301 с   5.84 МБ
--------------------------------------------------
Выполнено запросов   : 10/10
Среднее время запроса: 0.301 с
Скачано данных       : 58.40 МБ
Средняя скорость     : 19.40 МБ/с
```
